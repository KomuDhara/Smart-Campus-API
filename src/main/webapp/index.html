🏫 Smart Campus Sensor & Room Management API

Apache Tomcat 9  |  Jersey 2.x (JAX-RS 2.1)  |  Java 11

Base URL

http://localhost:8080/smartcampus-api/api/v1

Endpoints

MethodPathDescription
GET /api/v1 Discovery / API info
GET /api/v1/rooms List all rooms
POST /api/v1/rooms Create a room
GET /api/v1/rooms/{roomId} Get a specific room
PUT /api/v1/rooms/{roomId} Update a room
DELETE/api/v1/rooms/{roomId} Delete a room (409 if sensors assigned)
GET /api/v1/sensors List sensors (optional ?type=CO2)
POST /api/v1/sensors Register a sensor
GET /api/v1/sensors/{sensorId} Get a specific sensor
PUT /api/v1/sensors/{sensorId} Update a sensor
DELETE/api/v1/sensors/{sensorId} Delete a sensor
GET /api/v1/sensors/{sensorId}/readings List all readings for sensor
POST /api/v1/sensors/{sensorId}/readings Add a reading (403 if MAINTENANCE)
GET /api/v1/sensors/{sensorId}/readings/{id} Get a specific reading
ℹ️ Quick test: Open api/v1 to verify the API is running.
