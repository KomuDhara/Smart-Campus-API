POST /api/v1/rooms Create a new room ▼
Full URL
http://localhost:8080/Smart-Campus-API/api/v1/rooms
Postman Steps
  1. 1.Method: POST
  2. 2.URL: http://localhost:8080/Smart-Campus-API/api/v1/rooms
  3. 3.Headers: Content-Type: application/json
  4. 4.Body → raw → JSON (see below) — click Send
Request Body
{ "id": "R1", "name": "Lecture Hall A", "capacity": 100 }
Expected Responses
201Room created — returns room object
